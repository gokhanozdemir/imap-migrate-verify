# Security policy

Yandex Escape handles mailbox credentials and private message metadata. Do not
include either in public issues, pull requests, screenshots, or test fixtures.

## Reporting a vulnerability

Please use GitHub's private vulnerability reporting for this repository:
<https://github.com/gokhanozdemir/email-migration-verify/security/advisories/new>

Include the affected version, impact, and a minimal reproduction with all
credentials and personal data removed. Please do not open a public issue until
a fix is available.

## Supported version

Security fixes are applied to the latest version on the default branch.
